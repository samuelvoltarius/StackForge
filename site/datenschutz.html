Hosting & Server-Logfiles

Die Website wird auf einem Server der Hetzner Online GmbH (Industriestr. 25, 91710 Gunzenhausen, Deutschland/EU) als Auftragsverarbeiter betrieben. Beim Aufruf werden vom Webserver automatisch sogenannte Server-Logfiles erstellt, die folgende Daten enthalten können:

Lokale Speicherung im Browser (kein Cookie)

Diese Website verwendet keine Cookies, kein Tracking und keine Analyse-Tools. Gespeichert wird ausschließlich Ihre gewählte Sprache (Deutsch/Englisch) im localStorage Ihres Browsers, damit die Einstellung beim nächsten Besuch erhalten bleibt. Dieser Eintrag verbleibt nur auf Ihrem Gerät, wird nicht an uns oder Dritte übertragen und ist für diese Funktion technisch notwendig (daher einwilligungsfrei). Sie können ihn jederzeit in den Browser-Einstellungen löschen.
